Convert this eraser.io ERD to PostgreSQL SQL with the following specifications
1. Use PostgreSQL with pgcrypto and vector extensions 
2. All UUIDs should use gen_random_uuid() as default 
3. Use TIMESTAMPTZ DEFAULT now() for all timestamps
4. Use this exact syntax for the fts column: fts tsvector GENERATED ALWAYS AS (to_tsvector('english', content)) STORED 
5. Use the TEXT datatype instead of string
6. Use inline foreign key syntax only: column_name TEXT NOT NULL REFERENCES table(column) ON DELETE CASCADE
7. Add exactly these indexes with these names. Do not add any additional indexes beyond these two:
   - CREATE INDEX document_chunks_fts_idx ON document_chunks USING gin (fts);
   - CREATE INDEX document_chunks_embedding_hnsw_idx ON document_chunks USING hnsw (embedding vector_ip_ops);
